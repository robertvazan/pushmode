PushMode is a server-side Java library that streams web app's HTML output down to the browser while user's actions are streamed back to the server. See the website for more information.

The main package {@link com.machinezoo.pushmode} contains core classes, especially {@link com.machinezoo.pushmode.PushPage}. Package {@link com.machinezoo.pushmode.dom} contains classes that represent or construct DOM tree nodes. @see PushMode website
